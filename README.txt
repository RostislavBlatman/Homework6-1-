Программа "Мой блокнот"

 * Предназначена для добавления, удаления, редактирования и вывода записей из массива заметок, созданных в процессе
 * работы программы.
 * Разбита на три класса: Main - основной класс, Logic - класс с логикой приложения, Test - класс-тестировщик.
 * При запуске пользователю предоставляется выбор из 5 функций:
 * 1 - добавление записи ;
 * 2 - удаление записи   ;
 * 3 - редактирование    ;
 * 4 - вывод на экран    ;
 * 5 - выход             .
 * При выборе функции:
 * 1 - ваша заметка помещается в конец массива и он увеличивается на 1;
 * 2 - выбранная вами заметка удаляется и массив уменьшается на 1;
 * 3 - выбранная вами запись заменяется на новую, введённую непосредственно после приглашения к вводу;
 * 4 - выводит все записи, доступные на данный момент;
 * 5 - закрывает приложение.
